## 🎯 测试准备

### 1. 启动服务

```bash
cd /Users/xiaochenwu/Desktop/Agent-V3

# 方式1: 使用启动脚本
./start_all.sh

# 方式2: 手动启动
# 后端
python api_server.py --host 0.0.0.0 --port 8000 &

# 前端
cd frontend && pnpm dev &
```

### 2. 访问地址

- 前端: http://localhost:3000
- 后端API: http://localhost:8000
- API文档: http://localhost:8000/docs
